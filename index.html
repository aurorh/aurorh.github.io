Seattle Energy Map Project

This is a data visualization project I did in Jupyter Notebook mapping the energy use of each non-residential and multi-family building in Seattle. These are the two data sets I used:

Seattle Building Energy Benchmarking

Building_Energy_Benchmarking has stats on many energy use benchmarks for most big-ish (>20,000 sqft) buildings in the city, as well as each building's address, ZIP Code, year-built, ect. The statistic I'm using for this map is the SiteEUI(kBtu/sf), which they describe as: "Site Energy Use Intensity (EUI) is a property's Site Energy Use divided by its gross floor area. Site Energy Use is the annual amount of all the energy consumed by the property on-site, as reported on utility bills. Site EUI is measured in thousands of British thermal units (kBtu) per square foot." I chose this statistic because it wasn't missing very much data and it is based on square-footage, so large buildings won't skew the data.

Seattle GeoData ZIP Codes

This is a GeoJson file mapping out King County. It has King County sectioned off by ZIP Code which will match up nicely with the ZIP Codes in the CSV file.

My goal with this project was to use the CSV file to average out the data per building of each ZIP Code, then use those figures to create a heat map with the GeoJson file, and all to be done with Python code. Eventually I landed on using Jupyter Notebook because of the ability to split chunks of the code into "cells", which could be run one step at a time.
